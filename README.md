# Woodshed

Daily reps for technical interviews. A private study app: pattern-first concepts with
ELI5 explanations, worked JavaScript examples, 73 curated LeetCode problems, a daily
rep picker with streaks, and an interview-skills playbook.

Built with React, Vite, and Tailwind. The static build lives in `/docs` and is served
by GitHub Pages. Progress is stored in the browser (localStorage) with a sync-code
feature for moving it between devices.

Local dev: `npm install && npm run dev`. Rebuild after changes: `npm run build`.
